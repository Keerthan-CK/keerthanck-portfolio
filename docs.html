Technical Project Showcase

Predictive Maintenance for Automobiles

An IoT & ML-based system to predict failures in automobiles and reduce downtime by real-time sensor monitoring.

Built with ESP32, vibration sensors, MQ-3 gas sensors, current & voltage sensors, and temperature modules.

Data is processed via Flask on a central laptop, stored with SQLite, and visualized on a custom dashboard.

Alerts and anomaly detection are handled via Telegram bot integration and machine learning predictions.

Multi-Purpose Smart Student Card

Developed an RFID-based smart card system for campus automation, using ESP8266 and Django backend.

Enabled functions like lab inventory tracking, cloud document access, student profile management, and attendance automation.

Data stored securely on a Django server (PythonAnywhere) with Telegram bot integration for document download and alerts.

Water Quality Monitoring System

Real-time monitoring of turbidity, pH, TDS, temperature, and humidity using ESP8266.

Integrated OLED display for onsite data and Google Sheets logging via Wi-Fi for remote access.

Low-cost, low-power setup ideal for rural, agricultural, or environmental deployments.
